A very small setup for using Codex, Claude, or similar AI tools with shared Overleaf writing projects. Collaborators can keep writing in Overleaf as usual. The AI agent installs the shared instructions and saves project-history logs automatically.

How to Use This Guide

  1. Enable Overleaf Dropbox integration so the shared project is available as an existing folder for Codex or Claude.
  2. Install the shared instructions by copying the Codex or Claude setup prompt and pasting it into the tool. The AI agent should save the global instruction files automatically.
    Codex Setup Claude Setup
  3. Choose the existing Overleaf project folder when creating or opening a Codex/Claude project. No Dropbox reorganization is needed.
  4. Prompt normally. Ask the AI agent to edit, inspect, or revise in whatever way fits the task. The history file is the agent's responsibility.

Project History Convention

The AI agent should create or append to this file automatically:

project_history/Your_Name/YYYY-MM-DD by Agent.md

Use one folder per human collaborator and one file per date and AI agent.
