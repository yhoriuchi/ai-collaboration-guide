# Suggested Project Organization

Use clear project names so AI tasks are easy to find later.

Suggested prefixes:

```text
A | Administration or service work
T | Teaching
S | Software, packages, templates, guides
R | Research projects / papers
```

Examples:

```text
A | FSU Administration
T | 2026 Fall - Research Design
S | projoint
S | LaTeX Template
S | Replication Package Template
R | Example Research Project
```

For individual task titles, include date, model, effort level, and a short task description.

Examples:

```text
2026-07-15, GPT5.6 Terra, XHigh: Revise appendix
2026-07-15, GPT5.5, XHigh: Clean BibTeX entries
2026-07-15, Claude Opus, High: Check citations
```
